How to use

//create mnist object
var mnist = jMnist({
    trainCount: trainCount,             //integer (60000 is max and also default value)
    testCount: testCount,               //integer (10000 is max and also default value)
    trainImageFile: trainImageFile,     //file object
    trainLabelFile: trainLabelFile,     //file object
    testImageFile: testImageFile,       //file object
    testLabelFile: testLabelFile,       //file object
});

//start train and test
mnist.start(
    onTrainImageRead,                   //function(index, label, imageData)
    onTestImageRead                     //function(index, label, imageData)
);

"onTrainImageRead" and "onTestImageRead" is callback function. And these function have 3 parameters.

  • "index" is loop index.
  • "label" is teacher value.
  • "imageData" is 28*28 array, and each cell has 0 to 255 value.
