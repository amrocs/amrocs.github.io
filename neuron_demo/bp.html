How to use

neuron

A neuron.

group

Neurons grouped for some purpose. Each layers in Back Propagation are "group", and Self Organizing Map(SOM)'s neurons are also "group".

group.applyFunc( function )

Apply function to each neuron in group. For example, fire-function.
